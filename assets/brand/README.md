# NAVI brand

Canonical mark and demo assets for **NAVI** — coding agent engine; name puns on *nave* (spaceship).

## Concept (logo)

- Side-profile futuristic craft with cockpit canopy
- ~70° climb, deep-space field, cyan accents

## Files

| File | Role |
| --- | --- |
| **`navi-icon.jpg`** | Canonical logo |
| **`navi-demo.gif`** | Real TUI demo (README) — free model, tools, answer |
| **`navi-demo.mp4`** | Same recording, MP4 |
| **`navi-demo.tape`** | Charm [VHS](https://github.com/charmbracelet/vhs) script |

## Recording

Pure TTY capture of the real NAVI TUI (no mouse overlay).

```bash
# needs: vhs, navi binary
just record-demo
# or: bash scripts/record-demo.sh
```

Uses a **free** model (`opencode` / `mimo-v2.5-free`). Never Charm Hyper for the public GIF.
